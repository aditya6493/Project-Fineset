# How to get PDF or Word from the documentation

## Files

| File | Best for |
|------|----------|
| `docs/FINESET_COMPLETE_DOCUMENTATION.md` | Editing in VS Code / GitHub |
| `docs/FINESET_COMPLETE_DOCUMENTATION.html` | **PDF or Word** (recommended) |
| `ARCHITECTURE_GUIDE.md` | Shorter architecture reference |

---

## Option A — Save as PDF (easiest)

1. Open `docs/FINESET_COMPLETE_DOCUMENTATION.html` in **Chrome** or **Edge** (double-click the file).
2. Press **Ctrl + P** (Print).
3. Destination: **Save as PDF**.
4. Click **Save**.

---

## Option B — Open in Microsoft Word (.docx)

1. Open **Microsoft Word**.
2. **File → Open** → browse to `docs/FINESET_COMPLETE_DOCUMENTATION.html`.
3. **File → Save As** → choose **Word Document (*.docx)**.

---

## Option C — Google Docs

1. Upload `FINESET_COMPLETE_DOCUMENTATION.md` or `.html` to Google Drive.
2. Open with Google Docs.
3. **File → Download → PDF** or **Microsoft Word (.docx)**.

---

## Option D — VS Code extension

1. Install extension **"Markdown PDF"** in VS Code.
2. Open `docs/FINESET_COMPLETE_DOCUMENTATION.md`.
3. Right-click → **Markdown PDF: Export (pdf)**.
